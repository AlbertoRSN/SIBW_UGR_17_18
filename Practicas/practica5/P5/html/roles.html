- Modificar Rol de Usuario -

Funcion de cada Rol:

  1. Usuario Registrado: Podra incluir comentarios en cada obra del museo y cambiar sus datos de usuario
  2. Moderador: Podra editar / borrar comentarios.
  3. Gestor del Museo: Podra editar / borrar / anadir obras al museo, categorıas de las obras, etc.
  4. Superusuario (mínimo tiene que existir 1): Podra realizar cualquier accion, incluyendo modificar los roles de los usuarios.
